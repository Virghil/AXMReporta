# AXMReporta

## 📌 Descripción
AXMReporta es una aplicación web que permite a los ciudadanos reportar problemas de infraestructura urbana en la ciudad de Armenia, facilitando la gestión por parte de las entidades públicas.

## 🎯 Objetivo
Mejorar la comunicación entre ciudadanos y la administración pública mediante una plataforma digital accesible.

## ⚙️ Funcionalidades
- Registro e inicio de sesión de usuarios
- Recuperación de contraseña
- Creación de reportes con categoría
- Visualización de reportes
- Persistencia de datos con LocalStorage

## 🧠 Tecnologías
- HTML
- CSS
- JavaScript
- LocalStorage

## 🚀 Cómo ejecutar
1. Abrir el proyecto en Visual Studio Code
2. Ejecutar con Live Server
3. Ingresar por login.html
